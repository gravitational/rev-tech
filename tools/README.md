# tools

Helper scripts for repo-wide automation and CI enforcement.

| Script | Description |
|---|---|
| `require-readme.sh` | Enforces that every new directory added in a PR contains a README. Used in CI. |
| `teleportinfolabels` | Utility for managing Teleport info labels. |

For Terraform template validation tools, see [`templates/teleport-terraform/tools/`](../templates/teleport-terraform/tools/README.md).
